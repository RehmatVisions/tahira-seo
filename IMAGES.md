## üé® Image Optimization Tips

### 1. Compression
- Use tools like TinyPNG or ImageOptim
- Maintain quality while reducing file size
- Target 80-90% quality for JPGs

### 2. Format Selection
- **JPG:** Photos, certificates, complex images
- **PNG:** Logos, icons, images with transparency
- **WebP:** Modern format for better compression (optional)

### 3. Responsive Images
- Consider creating multiple sizes for different screen resolutions
- Use `srcset` attribute for responsive images

### 4. Alt Text
- All images have descriptive alt text
- Important for SEO and accessibility
- Describe the content and context
